關卡共同注意點：
CSS 命名語意化

外層 .wrap 可以設定 width, margin: 0 auto 讓版面水平置中

連結都要記得加上 hover 效果，以提升使用者體驗哦

區塊盡量不要寫死高度，如需留白可以使用 padding

不要直接在 HTML 標籤上設定 CSS 樣式，
因為一個網頁上有很多 h2, p，
若寫死的話之後會需要互相覆蓋比較麻煩且不好管理。

元素本身就是由上到下排列，
所以內層（標題、內文）的區塊就不需要設定 flex。
例如：第一關的三個區塊、第五關的中間區塊

部分關卡重點：
關卡五：

第三個區塊建議使用 row-reverse，這樣就不需要調整 HTML 結構順序
關卡十：

甜點類別非列表內容，請不要包在 ul li 中
